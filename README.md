## Projects

| Project | Description | Tools & Techniques |
|----------|--------------|--------------------|
| [**Bike Rental Database Management System**](./database-design) | Designed and implemented a relational schema for a simulated bike rental system using Oracle SQL, with functions, triggers, and stored procedures to enforce data integrity. | Oracle SQL, Data Modeling, Database Design |
| [**Forecasting U.S. Egg Prices (April 2025)**](./forecasting-egg-prices) | Forecasted U.S. egg prices using Linear Regression, Random Forest, Neural Network, and ARIMA models. Selected Linear Regression for best interpretability and context alignment. | R, Time Series Forecasting, Feature Engineering |
| [**Breast Cancer Classification using Machine Learning Models**](./breast-cancer-classification) | Compared Logistic Regression, SVM, Random Forest, Neural Network, and GBM models to classify tumor malignancy using R. Random Forest achieved the best balance across metrics. | R, Machine Learning, Classification |
| [**Phishing Email Detection Using Machine Learning Models**](./phishing-email-detection-using-ml) | Compared Logistic Regression and random Forest models to detect phishing emails using dataset of 18,650 using R. Applied TF-IDF text preprocessing and evaluated model performance using accuracy, precision, recall, and F1-score, with Random Forest achieving the best performance at 95.4% accuracy.| R,  Text Mining, Machine Learning, Classification |
